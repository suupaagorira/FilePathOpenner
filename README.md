# FilePathOpenner

FilePathOpenner は、クリップボードにコピーしたパスをショートカットキーで素早く開くための Electron 製アプリです。コピーしたテキストを整形してファイル/フォルダや URL を開き、Windows ではスタートアップへの登録もサポートします。

## Features

- **Clipboard Path Opening**: ショートカットでクリップボード上のパスを開きます。複数行なら改行区切りでそれぞれ開くことも、1 行に結合して開くことも可能です。
- **Parent Directory Option**: 別のショートカットで 1 階層上のディレクトリ（または URL の親）を開けます。
- **Input Cleanup**: 前後の空白除去や指定文字の削除など、パス整形オプションを備えます。
- **Base Path Prefix**: GUI 上で前提パスを設定し、相対パス検索時に先頭へ付加できます。フォルダ選択ダイアログからの指定にも対応します。
- **Prefix Rules**: 「DOC-123」のような ID を URL や共有フォルダのパスへ変換するルールを、優先順位付きで登録できます。
- **Path Fallback**: 指定したパスが存在しない場合、親ディレクトリを自動で開き、元のパスと移動した階層数を知らせます。
- **Windows Startup Registration**: Windows 環境ではスタートアップフォルダにショートカットを作成・削除できます。

## Settings UI

設定画面はサイドバーナビゲーション（使い方 / ショートカット / パスの整形 / 変換ルール / 動作テスト / 一般）で構成されています。

- **自動保存**: すべての設定は変更した時点で保存され、トーストで通知されます。「設定」ボタンはありません。
- **ショートカットレコーダー**: 入力欄をクリックしてキーをそのまま押すだけで割り当てられます。Esc でキャンセル、Backspace で無効化。他アプリと競合して登録に失敗した場合はその場で警告が表示されます。
- **動作テスト**: 実際に開く前に、コピー予定のテキストが現在の設定でどう解釈されるか（URL / ファイル / フォルダ / 親フォルダへのフォールバック / 見つからない）をドライランで確認できます。
- **使い方ガイド**: 初めてのユーザー向けに、3 ステップの使い方と対応入力の一覧をアプリ内に表示します。
- **トレイ常駐**: ウィンドウを閉じてもタスクトレイに常駐します（初回のみ通知）。終了はトレイメニューから。多重起動は自動的に防止されます。

## Build Instructions

1. 依存関係をインストールします。

   ```bash
   npm install
   ```

2. アプリを起動する場合は次のコマンドを実行します。

   ```bash
   npm start
   ```

3. ポータブル版をビルドするには以下を実行します。成果物は `release/` ディレクトリに生成されます。

   ```bash
   npm run dist
   ```

## Known Limitations

- スタートアップ登録機能は Windows のみに対応しています。
- 他 OS ではパスのオープン自体は動作しますが、細かな動作差異がある可能性があります。
