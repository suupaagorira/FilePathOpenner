# ユーザーマニュアル

## 概要
FilePathOpenner は、クリップボードにコピーしたパス（ファイル、フォルダ、URL）を素早く開くための Electron 製アプリです。あらかじめ設定したショートカットキーでクリップボードの内容をパースし、存在するパスや URL を開きます。

## 対象ユーザー
- Windows でショートカットを使ってパスを瞬時に開きたいユーザー
- クリップボードからのパス処理や開閉操作を自動化したいエンドユーザー

## 起動方法
1. 依存モジュールをインストール（開発環境）

```pwsh
npm install
```

2. アプリを起動

```pwsh
npm start
```

### ポータブルや配布版を使う場合
- `release/` フォルダ内に配布物がある場合は、該当する exe/portable を実行します。

## メイン機能の使い方
- クリップボードからパスを開く
  - GUI で設定したショートカット（デフォルト `Ctrl+E`）を押すと、クリップボードにあるパスが開きます。URL は外部ブラウザで開かれます。複数行ある場合は、改行ごとにそれぞれ開くか、1行に結合して開くかを `設定` で切り替えできます。

- 一つ上の階層を開く
  - `Ctrl+Shift+E`（デフォルト）などで、URL やパスの 1 階層上を開きます。

- `Base Path` の設定
  - 相対パスを扱う際に先頭に付加される基準パスを設定できます。GUI 上で設定し、保存すると即時反映されます。

- プレフィックス判定ルール
  - コピーした文字列が、設定した「先頭パターン」で始まるとき、その文字列を URL や共有フォルダの判別パス部とみなし、「結合文字列」と連結して開きます。
  - 例 1（URL）: 先頭パターン `DOC-` ／ 結合文字列 `https://intra/docs/` → `DOC-123` をコピーして `Ctrl+E` を押すと `https://intra/docs/DOC-123` を外部ブラウザで開きます。
  - 例 2（共有フォルダ）: 先頭パターン `案件` ／ 結合文字列 `\\server\share\` → `案件A\資料` をコピーすると `\\server\share\案件A\資料` を開きます。
  - 「先頭パターンを除去して結合」にチェックすると、先頭パターン部分を取り除いてから結合します（例: 先頭パターン `id:` ／ 結合文字列 `https://intra/view?id=` で `id:42` → `https://intra/view?id=42`）。
  - ルールは上から順に評価され、最初にマッチしたルールが適用されます。プレフィックスルールにマッチした場合は `前提パス` より優先されます。各ルールカードの「↑」「↓」ボタンで評価順を入れ替えられます。
  - 各ルールカードには、入力中の設定でどのように変換されるかのプレビューが表示されます。
  - 既に `http(s)://` の URL や絶対パスと判定される文字列にはルールを適用しません。

## 設定項目（GUI）
- クリップボードを開くショートカット: `openShortcut`（例: `Ctrl+E`）
- 1 階層上を開くショートカット: `openParentShortcut`（例: `Ctrl+Shift+E`）
- 1 行として開く: `openAsSinglePath`（改行を全て結合する）
- 文字列の前後空白を削除: `trimSpaces`
- 文章の前後から除去する文字リスト: `removeList`（例: `"'<>[]`）
- 前提パス: `basePath`（相対パス時に付加）
- プレフィックス判定ルール: `prefixRules`（`{ prefix, base, stripPrefix }` の配列。先頭パターン一致時に `base` と結合して開く）

## タスクトレイ動作
- ウインドウの「×」を押してもアプリは終了せず、タスクトレイに常駐します。
- タスクトレイのコンテキストメニューから `設定画面を開く` と `ソフトを終了する` が選べます。

## スタートアップ登録（Windows のみ）
- GUI から `登録する` を押すと、アプリのショートカットが Windows のスタートアップフォルダに配置されます。
- 削除は `解除する` を押します。

## よくある操作フロー
1. クリップボードに「C:\\Users\\user\\Documents\\file.txt」をコピー
2. `Ctrl+E` を押す → ファイルが関連アプリで開かれる
3. 相対パスを扱う場合は `basePath` を設定しておく

## 注意点
- `register-startup` は PowerShell を使って `.lnk` ショートカットを作成しているため、管理者権限やポリシーによって失敗することがあります。
- 1 行として扱う場合、改行が除去されるため複数パスは一つとして扱われます。
