# FAQ (よくある質問)

Q1: クリップボードのパスを開くショートカットが動作しない

- A: 「ショートカット」パネルでキーが設定されているか確認してください。設定は自動保存され、その場でグローバルショートカットが再登録されます。
- A: ショートカットが他のアプリと競合している場合、入力欄の下に「登録に失敗しました」と表示されます。別のキーに変更してください。

Q2: クリップボードにパスがあるのに何も起きない

- A: アプリがすでに `globalShortcut` を登録しているか確認して下さい。管理者権限や他アプリが同じショートカットを登録していると失敗することがあります。
- A: `openAsSinglePath` の設定によって改行が結合されている可能性があります。

Q3: URL をブラウザで開きたいが開かない

- A: `openExternal` は外部ブラウザを呼び出します。OS やブラウザの関連付け設定を確認してください。

Q4: スタートアップに登録できない

- A: PowerShell 実行のポリシーやアクセス権の問題でショートカットの作成が失敗する場合があります。管理者権限で操作を試すか、ポリシーを確認してください。

Q5: アプリを完全に削除したい

- A: Windows の場合は `release` のインストーラや配布方法に応じてアンインストール処理を実行してください。ユーザー設定は `electron-store` の保存場所を手動で削除する必要があります。

Q6: 思ったとおりに開けるか、事前に確認したい

- A: 「動作テスト」パネルにテキストを貼り付けるか「クリップボードを読み込む」で取り込むと、実際に開かずに解析結果（URL / ファイル / フォルダ / ○階層上を開く / 見つからない）を確認できます。整形・変換ルールなど現在の設定がそのまま適用されます。
